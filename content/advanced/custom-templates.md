# Wide Layout

This page uses the `wide` template defined in the frontmatter:

```yaml
---
template: wide
---
```

It removes the sidebar constraint to give you more breathing room for large diagrams or tables.

| Option | Type | Description |
| :--- | :--- | :--- |
| `template` | `string` | The layout component to use. |
| `title` | `string` | Page title for metadata. |
| `author` | `string` | Page author. |
